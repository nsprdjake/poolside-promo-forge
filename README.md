# Poolside Promo Forge

A self-contained single-file web app that generates retro Palm Springs-style promo packs for local businesses.

## What it does
- Builds a promo concept from a business name, offer type, audience, city, and vibe
- Generates a headline, offer angle, CTA, flyer copy, social caption, SMS blast, and hashtags
- Shows a polished poster mockup with dynamic colors
- Exports the poster as a PNG and supports copy-to-clipboard for every content block

## Why this one
It leans more productizable than pure whimsy: it could become a lightweight lead-gen / creative intake tool for Inspired Design or a micro-SaaS promo generator for local businesses.

## Files
- `index.html` — everything inline: HTML, CSS, and JavaScript
